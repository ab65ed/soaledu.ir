## 🔍 نکات مهم ساختار

### ✅ ویژگی‌های ساختار
- **Monorepo Structure**: ساختار یکپارچه
- **Type Safety**: TypeScript در همه جا
- **Separation of Concerns**: جداسازی منطقی
- **Scalable Architecture**: قابل گسترش
- **Documentation First**: مستندسازی کامل

### 📁 پوشه‌های کلیدی
- **`/frontend/src/app/`**: App Router Next.js 14
- **`/backend/src/`**: معماری MVC
- **`/docs/`**: مستندات فنی
- **`/report/`**: گزارش‌های تحلیلی
- **`/tests/`**: تست‌های جامع

### 🎯 الگوهای طراحی
- **Domain-Driven Design**: طراحی مبتنی بر دامنه
- **Repository Pattern**: الگوی مخزن
- **Service Layer**: لایه سرویس
- **Component-Based**: مبتنی بر کامپوننت

## 🚀 راهنمای استفاده

### فرانت‌اند
```bash
cd frontend/
npm install
npm run dev      # توسعه
npm run build    # بیلد
npm run test     # تست
```

### بک‌اند
```bash
cd backend/
npm install
npm run dev      # توسعه
npm run build    # بیلد
npm run test     # تست
```

### تست‌ها
```bash
npm run test:unit      # تست واحد
npm run test:e2e       # تست E2E
npm run test:coverage  # پوشش تست
```
